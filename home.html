Important FAQ

What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface that represents a web document as a tree of objects, allowing developers to access and manipulate HTML or XML content, structure, and styles dynamically, typically using JavaScript.

How do you select an element from the DOM?

To select an element from the DOM in JavaScript, you can use several methods: document.getElementById('id') selects an element by its ID, document.getElementsByClassName('class') selects elements by their class name, document.getElementsByTagName('tag') selects elements by their tag name, document.querySelector('selector') selects the first matching element using a CSS selector, and document.querySelectorAll('selector') selects all matching elements.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique in the DOM where a single event listener is attached to a parent element instead of individual child elements. When an event occurs on a child, it bubbles up to the parent, allowing the parent to handle the event. This is useful because it reduces memory usage, improves performance, and simplifies code maintenance by minimizing the number of event listeners needed.

How do you manipulate an element's attributes and styles using the DOM?

You can manipulate an element's attributes and styles in the DOM using JavaScript. To change attributes, use element.setAttribute('attribute', 'value') or directly access properties like element.src or element.href. To change styles, use element.style.property = 'value', for example, element.style.color = 'red'.

What is the role of the dataset property in the DOM, and how can it be used to manage custom data attributes?

The dataset property in the DOM allows you to access and manage custom data attributes (prefixed with data-). Each attribute is available as a camelCase property on the dataset object. For example, data-user-id can be accessed as element.dataset.userId. You can get or set values easily using this property.
